Array Problem Set

01. Write a function printReverse() that takes an array as an argument and prints out the elements in the array in reverse order (don't actually reverse the array itself).

02. Write a function isUniform() which takes an array as an argument and returns true if all elements in the array are identical.

03. Write a function sumArray() that accepts an array of numbers and returns the sum of all numbers in the array.

04. Write a function max() that accepts an array of numbers and returns the maximum number in the array.
